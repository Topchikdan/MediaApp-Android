## О проекте

Минималистичный аудиоплеер для Android с тёмным интерфейсом на базе почти чёрного фона с фиолетовым акцентом. Читает треки с внешнего хранилища устройства, не даёт экрану гаснуть во время воспроизведения.

---

## Возможности

- Воспроизведение локальных аудиофайлов с внешнего хранилища устройства
- Управление воспроизведением: кнопки «Играть», «Пауза» и «Стоп»
- Прогресс-бар: полоса перемотки с отображением текущего времени и длительности трека
- Выбор трека через выпадающий список всех доступных аудиофайлов
- Название текущего трека отображается по центру экрана
- Обложка альбома: стилизованный арт-блок с иконкой ноты
- WAKE_LOCK: экран не гаснет во время воспроизведения

## Быстрый старт

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Topchikdan/MediaApp-Android.git
cd MediaApp-Android
```

2. Откройте проект в **Android Studio**.
3. Подключите устройство или запустите эмулятор (API 21+).
4. Нажмите **Run ▶** или выполните:

```bash
./gradlew assembleDebug
```

5. При первом запуске выдайте разрешение на чтение хранилища.

---

## Требования

| Компонент | Версия |
|---|---|
| Android Studio | Hedgehog или новее |
| Min SDK | Android 5.0+ (API 21) |
| Target SDK | API 31 (Android 12) |
| JDK | 11+ |
| Аудиофайлы | Должны быть во внешнем хранилище устройства |

## Структура проекта

```
app/
└── src/main/
    ├── AndroidManifest.xml              # разрешения и точка входа
    ├── java/com/example/media/
    │   └── MainActivity.java            # логика плеера
    └── res/
        ├── layout/
        │   └── activity_media.xml       # разметка главного экрана
        ├── drawable/
        │   ├── album_bg.xml             # фон блока обложки
        │   ├── btn_primary.xml          # стиль кнопки «Играть» (фиолетовый круг)
        │   ├── btn_secondary.xml        # стиль кнопок «Пауза» и «Стоп»
        │   ├── circle_accent.xml        # акцентный круглый элемент
        │   └── dropdown_bg.xml          # фон выпадающего списка треков
        └── mipmap/
            └── ic_launcher/             # иконки разных плотностей
```

## Технологии

| Компонент | Описание |
|---|---|
| Java / Kotlin | Язык разработки |
| MediaPlayer | Воспроизведение аудио (Android SDK) |
| ConstraintLayout | Разметка экрана |
| READ_EXTERNAL_STORAGE | Чтение файлов с хранилища |
| WAKE_LOCK | Фоновая работа без гашения экрана |
| Gradle (Kotlin DSL) | Система сборки |

## Дизайн

Тёмная тема на базе почти чёрного фона с фиолетовым акцентом:

| Токен | Цвет | Назначение |
|---|---|---|
| Фон экрана | `#0F0F13` | Основной фон |
| Фон карточек | `#1E1E2E` | Блок обложки, дропдаун |
| Разделитель | `#2A2A3E` | Линии списка, неактивные элементы |
| Акцент | `#6C63FF` | Кнопка «Играть», прогресс SeekBar |
| Нажатый акцент | `#5A52CC` | Кнопка «Играть» при нажатии |
| Текст | `#FFFFFF` | Заголовки, название трека |
| Вторичный текст | `#888899` | Время, подписи кнопок |

## Разрешения

| Разрешение | Назначение |
|---|---|
| `READ_EXTERNAL_STORAGE` | Чтение аудиофайлов с памяти устройства |
| `WAKE_LOCK` | Экран не гаснет во время воспроизведения |
