# MediaApp Android

**Мобильное медиаприложение для воспроизведения и управления контентом на платформе Android.**

Удобный медиаплеер на **Kotlin** для просмотра и прослушивания медиафайлов прямо со смартфона — музыка, видео и всё любимое всегда под рукой. 🎵🎬

## ✨ Основные возможности

- Воспроизведение аудио и видеофайлов
- Удобный и интуитивный медиаплеер
- Управление медиатекой
- Поддержка популярных форматов
- Чистый и современный интерфейс

## 🛠 Технологии

- **Kotlin** — основной язык разработки
- **Android SDK**
- **Gradle (Kotlin DSL)** — система сборки
- Современная архитектура Android-приложений

## 🚀 Как запустить проект

# Клонировать репозиторий
git clone https://github.com/Topchikdan/MediaApp-Android.git

# Перейти в папку проекта
cd MediaApp-Android

Открыть проект в **Android Studio**, дождаться синхронизации Gradle и запустить на эмуляторе или физическом устройстве.

## 📋 Требования

- Android Studio Hedgehog или новее
- Android SDK 24+
- JDK 11+
